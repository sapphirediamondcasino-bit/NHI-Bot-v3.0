Bot Commands

/welcome
Opens an interactive configuration panel to customize your server's welcome system. Set welcome channel, choose themes, customize text, fonts, backgrounds, and preview welcome cards.
/premium
View premium tier information and purchase options. Displays available features for each tier and upgrade benefits.
/roleselector
Creates an interactive role selection panel with buttons. Server owners can customize which roles are available for self-assignment.
/ping
Shows bot statistics including latency, uptime, server count, and system performance metrics.
/ban @user [reason]
Ban a user from the server. Requires Ban Members permission. Sends a DM notification to the banned user with the reason.
/kick @user [reason]
Kick a user from the server. Requires Kick Members permission. User can rejoin with a new invite.
/timeout @user [duration] [reason]
Timeout a user for a specified duration. Prevents them from sending messages, adding reactions, or joining voice channels.
/warn @user [reason]
Issue a warning to a user. Warnings are logged and can be viewed by moderators. User receives a DM notification.
!welcome
Alternative prefix command for welcome system configuration. Opens the same interactive panel as /welcome.
!premium
View premium tier information using prefix commands.
!ban @user [reason]
Ban a user using prefix commands. Same functionality as slash command.
!kick @user [reason]
Kick a user using prefix commands.
!timeout @user [duration] [reason]
Timeout a user using prefix commands. Duration can be specified in minutes (m), hours (h), or days (d).
!warn @user [reason]
Issue a warning using prefix commands.
!ping
Check bot statistics and latency using prefix commands.
!roleselector
Create role selector panel using prefix commands.

Setup Guide for Server Owners

Step 1: Invite the Bot

Click the "Invite NHI Bot" button at the top of this page. Make sure to grant all requested permissions for full functionality.

Step 2: Configure Permissions

Ensure NHI Bot has the following permissions in your server:

  • Manage Roles (for role selector)
  • Manage Channels (for welcome channel setup)
  • Ban Members & Kick Members (for moderation)
  • Moderate Members (for timeout)
  • Send Messages & Embed Links (for welcome cards)
  • Attach Files (for welcome images)

Step 3: Set Up Welcome System

Use the /welcome command to open the configuration panel. Select your welcome channel, choose a theme, and customize your welcome message.

Step 4: Configure Role Selector

Use /roleselector to create self-assignable role buttons. The bot will guide you through selecting which roles members can self-assign.

Step 5: Test Your Setup

Use /ping to verify the bot is responsive. Test the welcome system by having a friend join or using the preview feature in the welcome configuration.

Troubleshooting

🚫 Bot Not Responding to Commands

Solution: Ensure the bot has "Use Application Commands" permission enabled in your server settings. Check that the bot role is not positioned below other roles that might restrict permissions.

📭 Welcome Messages Not Sending

Solution: Verify that the bot has "Send Messages" and "Attach Files" permissions in your welcome channel. Use /welcome to check if a welcome channel is properly configured.

🎭 Role Selector Not Working

Solution: Make sure the NHI Bot role is positioned ABOVE the roles you want members to self-assign. Discord's role hierarchy requires bots to be above roles they manage.

🔨 Moderation Commands Failing

Solution: Check that the bot has appropriate moderation permissions (Ban Members, Kick Members, Moderate Members). Ensure the bot's role is higher than the target user's highest role.

⚙️ Premium Features Not Available

Solution: Use /premium to verify your subscription status. If you've recently upgraded, try using the command again after a few minutes as tier updates may take time to sync.

🖼️ Welcome Card Images Not Loading

Solution: Ensure the bot has "Attach Files" permission. If using custom backgrounds (premium feature), verify your image URL is publicly accessible and in a supported format (PNG, JPG, GIF).

The Journey to v3.0

v1.0 - The Beginning (2020)

NHI Bot was originally created for a hockey server community. It started as a simple role selection bot under Kurrasch Networking Solutions. The bot provided basic functionality for members to self-assign roles and helped manage the growing hockey community.

The Shutdown (2022)

After two years of service, NHI Bot went offline. Technical issues and infrastructure changes made it difficult to maintain. When asked about the bot's status, delivering the news that "NHI Bot shut down 3 years ago" was heartbreaking - seeing your own work taken down is never easy.

The Rebirth - Kurrasch Systems (2023)

After long discussions with colleagues and buddies, Kurrasch Systems was born in 2023, marking the end of an era for Kurrasch Networking Solutions. This transformation brought new life and possibilities. Most to all bots are now powered through Kurrasch Systems, representing a unified and more powerful infrastructure.

The Merger - Kurrasch Gaming Merch Store LLC (Fall 2024)

Kurrasch Systems joined forces with Kurrasch Gaming Merch Store LLC in fall 2024. The LLC, which creates custom gaming merchandise and items, had just gotten their store settled when they decided to take on the challenge. "We scaled too quick," but the LLC was ready: "bring it on!" The LLC bought the old network, and it became what it is today - a complete ecosystem of services.

v2.0 Beta - Development Phase (Early 2024)

Development resumed on NHI Bot after 2 years of dormancy. All recoverable files from the old bot system were salvaged and rebuilt from the ground up. The beta introduced the foundation for the welcome system, moderation tools, and began implementing features found in popular Discord bots.

v3.0 - Full Release (October 2024)

NHI Bot v3.0 represents a complete transformation from its humble beginnings. What started as basic role selection has evolved into a feature-packed bot with:

  • 4-tier premium system with extensive customization
  • 240 seasonal themes across 12 holiday packs
  • Advanced welcome card system with 9 base color themes
  • Complete moderation suite (ban, kick, timeout, warn)
  • Interactive role selector with button controls
  • Custom fonts, backgrounds, avatar frames, and animated effects
  • Both slash commands and prefix command support
  • Persistent settings and error handling

This bot is jam-packed with features that rival popular bots on Discord. From the ashes of the 2022 shutdown, NHI Bot has risen stronger than ever, now backed by the full power of Kurrasch Systems and the Kurrasch Gaming Merch Store LLC.
